# StandsolutionsNFT contracts

## How to start

```shell
$ https://github.com/Stand-Solutions/NFT
$ cd StandsolutionsNFT
$ npm i
```

### Build contracts

```shell
$ npm run build
```

### Run unit tests

```shell
$ npm run test
```

### Deploy contract

```shell
$ npm run deploy
```

### Verify contract
```shell
$ npm run verify contract_address
```
